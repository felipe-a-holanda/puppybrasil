Welcome, woof, woof!
As this is the first time that you have started Puppy, he is running totally in RAM!
RAM space A readout in the taskbar shows how much working space left in RAM*. If less than 30MB, space is critical, so do not run big apps like SeaMonkey -- instead, shutdown Puppy and you will be asked to create a permanent storage file, that will give you more space.
Internet connection
Click on the 'connect' icon at left of the screen (one click only -- do not double-click!). You will then see some buttons for choosing how you wish to connect to the Internet -- click the most appropriate. Configuration of Puppy for Internet access will in most cases be very straightforward.
'connect' icon

I need help! Lots of local help is available -- select 'Help' in the menu (see left side of taskbar). The local Help page also has the Release Notes for this version of Puppy -- well worth checking out!
When you get online, the web browser home page has many more links.

...notice also, the 'Shutdown' entry in the menu. For an older RAM-challenged PC (128MB or less), recommend reboot now to create more working space!
Saving a session At first shutdown, you are asked where you want to save your personal files/data/settings to, and this can be any FAT (as used for MSDOS or Windows 9x) partition, NTFS (as used by Windows XP) partition or Linux partition in the hard drive, or any plugin drive (USB Flash pen drive for example).**


*   Note, some builds of Puppy have the free memory shown directly in the tray, most recent puppies have an icon that looks kind of like an integrated circuit, with a green/red bar to indicate % memory -- in the latter case, do a mouse-over to read exact free memory (ditto for CPU temperature, battery charge, etc).
** Note, a 'partition' is known by a drive letter in dos/windows, such as C:, D:, but Linux uses a more rigorous naming scheme, such as sda1, sda2. If you are unfamiliar with these different types of partitions (and the Linux naming), don't worry, as Puppy will automatically detect and offer you a choice of suitable partitions in which to save the session. Puppy will only create a single file in the partition and this will not affect your current installation of Windows (but of course we have to make a full disclaimer of any responsibility if something does go wrong!) and next time you boot Puppy from CD the save-file will get automatically read -- so, there is really no need to install Puppy onto the hard drive, and you can just keep booting Puppy from CD indefinitely.
Note, the jargon term we use to refer to this file is it is the "pupsave" file.
