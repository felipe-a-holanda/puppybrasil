InkLite


Light version of Inkscape vector graphics editor, especially for Puppy.

InkLite is based upon Inkscape v0.36, the last release of Inkscape that was written in C (not C++) and did not use the GTKMM library. InkLite is about 1/5 the size of the latest Inkscape.

Inkscape was originally a fork from the Sodipodi project, and the documentation for Sodipodi is the closest to the features offered in InkLite.

Sodipodi home page: http://www.sodipodi.com/index.php3
Online documentation for Sodipodi: http://www.sodipodi.com/index.php3?section=documentation
Online documentation for Inkscape (later versions than InkLite): http://www.inkscape.org/doc/index.php
